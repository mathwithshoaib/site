📸 Photo Gallery — How to Add Your Photos

To add a photo gallery to this page, follow these steps:

  1. Create an images/gallery/ folder inside your site folder on your computer
  2. Add your photos (JPG or PNG) into that folder — name them simply: photo1.jpg, photo2.jpg, etc.
  3. Upload the entire images/gallery/ folder to your GitHub repository (same way you uploaded the other files)
  4. Tell me how many photos you added and I will write the gallery code for this page — it will display them in a beautiful grid with a lightbox viewer
Tip: You can add photos from France, LUMS, GCU, Math Circles events, or anywhere else. The gallery can be organised by category if you like.
